# AI-Microscope-Blood-Cell-Detection-System
## About

This project builds an AI system that automatically detects and classifies 
blood cells in microscopic images. Using object detection techniques, the 
system identifies three types of cells — Red Blood Cells (RBC), White Blood 
Cells (WBC), and Platelets — then draws bounding boxes and counts each type.

The goal is to assist medical professionals by reducing the time and effort 
required for manual blood cell analysis, while improving consistency and accuracy.
